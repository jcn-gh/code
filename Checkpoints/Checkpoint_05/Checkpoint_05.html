¿Qué es un condicional?

Es la forma que tienen los lenguajes de programación de hacer que se ejecuten unas instrucciones u otras dependiendo de si unas condiciones se cumplen o no. Es decir, se toman decisiones teniendo en cuenta las condiciones establecidas.

En Python, se usan las palabras clave 'if' (si), 'else' (sino) y 'elif' (contracción de else if) para definir las diferentes situaciones que nos podemos encontrar. La palabra clave «if» se utiliza para la primera condición, «elif» se utiliza para condiciones adicionales y «else» se utiliza para el caso en que ninguna de las condiciones anteriores sea verdadera. Esta es la sintaxis básica:

if condición_1:
    # código que se ejecuta si condición_1 es verdadera
elif condición_2:
    # código que se ejecuta si condición_1 es falsa pero condición_2 es verdadera
else:
    # código que se ejecuta si ninguna de las condiciones anteriores es verdadera

Por ejemplo, podríamos tener un programa que imprima si un número es positivo, negativo o cero:

numero = input("Por favor, introduzca un número: ")
numero = int(numero)

if numero > 0:
    print("El número es positivo.")
elif numero < 0:
    print("El número es negativo.")
else:
    print("El número es cero.")

# instrucciones fuera de las condiciones

Resaltar que cada instrucción condicional finaliza con dos puntos ":" y que las instrucciones a ejecutar deben llevar una sangría o indentación.

Se pueden incluir condicionales dentro de otros, dando lugar a lo que se llaman condicionales anidados.

password = input("Introduzca contraseña: ")

if (len(password) >= 8):
    print('La contraseña es suficientemente larga.')

    if(password == 'miClaveSegura'):
        print("Además es la correcta.")
    else:
        print("Pero es incorrecta.")
elif (password != 'miClaveSegura'):
    print('Además de incorrecta, la contraseña es muy corta e insegura.')

Así pues, los condicionales nos permiten definir en nuestro programa múltiples vías de ejecución, contemplando así todas las posibilidades de un proceso. Ten en cuenta que, Muchas veces, elegir qué hacer implica una combinación de varias condiciones para tomar la decisión final.

¿Cuáles son los diferentes tipos de bucles en python? ¿Por qué son útiles?

Los bucles, también conocidos como ciclos permiten repetir bloques de código muchas veces sin tener que escribir el mismo código una y otra vez. Python tiene dos tipos de bucles o ciclos: el bucle 'for' (para) y el bucle 'while' (mientras).

  1. Bucle For: Este tipo de bucle se utiliza para recorrer una secuencia (como una lista, tupla, diccionario, conjunto o cadena) o cualquier otro objeto iterable (repetitivo). La sintaxis básica es:

    for variable in secuencia: # código a ejecutar para cada
print('Fin del programa.')

    el ciclo se ejecuta interminablemente. Sin embargo, al haber introducido la condición de salida (num == 0) nos aseguramos que, en ese momento, se cierre el ciclo.

Los bucles son útiles en programación porque permiten ejecutar un bloque de código repetidamente. Por ejemplo, podemos usar un bucle "for" para procesar todos los elementos de una lista, o un bucle "while" para repetir una acción hasta que se cumpla una condición (como esperar a que el usuario introduzca una entrada válida). Sin embargo, desde el punto de vista de la eficiencia, es recomendable evitarlos en la medida de lo posible. Siempre que vayas a usar un ciclo deberías preguntarte si es totalmente necesario, si puedes recudir la cantidad de veces que vaya a repetir las instrucciones o si existe una manera más efectiva de hacerlo.

¿Qué es una comprensión de listas en python?

La comprensión de listas es una característica única y poderosa de Python que permite crear y manipular listas de manera eficiente con menos código, aplicando una expresión a cada elemento de una secuencia iterable. Esencialmente, es una forma de transformar una lista en otra lista, aplicando una expresión a cada uno de sus elementos, y filtrando los resultados.

La sintaxis básica de una comprensión de listas es:

nueva_lista = [expresión for elemento in lista_original if condición]

La "expresión" es el resultado que queremos obtener y se aplica a cada elemento de la lista. La parte "for elemento in lista_original" es similar a un bucle for normal en Python donde "elemento" es una variable temporal que toma cada valor de la lista original durante la iteración. La "condición" es opcional y se puede usar para filtrar los elementos que se incluirán en la nueva lista. En esencia: "haz esto para todos los elementos de la lista que cumplan la condición".

Por ejemplo, si queremos crear una lista de los cuadrados de todos los números del 1 al 10, podríamos usar una comprensión de listas de la siguiente manera:

cuadrados = [x**2 for x in range(1, 11)]
        print(cuadrados)

Esto imprimirá: [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

En este caso, "x**2" es la expresión que se aplica a cada número, "x" es la variable que toma cada valor de la lista números del 1 al 10.

Si tenemos una lista de autores y queremos otra con aquellos que comienzen por "j" y ponerlos en mayúsculas, podríamos usar una comprensión de listas de la siguiente manera:

autores = ["jane austen","george orwell","james clear","cal newport"]

autor_lista = [autor.title() for autor in autores if autor.startswith('j')]
print(autor_lista)

Esto imprimirá: ["Jane Austen", "James Clear"]

En este ejemplo, para poner en mayúscula cada autor se aplica .title(), autor es la variable que toma cada valor de la lista autores, y se aplica .startswith('j') a cada autor para saber si comienza por 'j'.

Las comprensiones de listas son una forma muy eficiente y legible de realizar operaciones comunes en listas, y son una de las características que hacen a Python un lenguaje tan popular para el procesamiento de datos.

¿Qué es un argumento en Python?

Un argumento en Python es un valor que se pasa a una función o método cuando se llama. Los argumentos se utilizan para proporcionar datos a las funciones y permitirles realizar operaciones con esos valores de entrada.

Por ejemplo, la siguiente función suma dos números:

def suma(a, b):
    return a + b

Aquí, 'a' y 'b' son argumentos de la función 'suma'. Cuando llamamos a la función, debemos proporcionar dos argumentos:

resultado = suma(6, 3)
print(resultado) # Imprime: 9

resultado = suma(b=30, a=10)
print(resultado) # Imprime: 40

En este caso, los números 6 y 3 son los argumentos que pasamos a la función 'suma'.

Python soporta argumentos por defecto, lo que significa que puedes asignar un valor predeterminado a un argumento que se utilizará si no se proporciona un valor al llamar a la función. Por ejemplo:

def saludar(nombre="extraño"):
    print(f"Hola, {nombre}!")

    saludar("Alice")  # Imprime "Hola, Alice!"
    saludar()  # Imprime "Hola, extraño!"

Aquí, "nombre" es un argumento con un valor predeterminado de "extraño". Si no proporcionamos un argumento al llamar a la función "saludar", se utilizará el valor predeterminado.

En Python, es posible llamar a una función proporcionando explícitamente el nombre de los argumentos junto con sus valores. Esto puede hacer que el código sea más fácil de leer, ya que queda claro qué argumento representa cada valor.

La siguiente función que calcula el área de un rectángulo:

def area_rectangulo(ancho, alto):
        return ancho * alto

Podríamos usar argumentos de palabras clave para hacer que el código sea más claro:

a = area_rectangulo(ancho=5, alto=3)

Así queda claro que el primer argumento representa el ancho del rectángulo y el segundo argumento representa su alto.

Quizá en alguna ocasión no sabemos de antemano cuantos elementos vamos a enviar a una función. En estos casos podemos utilizar los parámetros indeterminados por posición y por nombre.

Para manejar un número indeterminado de argumentos posicionales, podemos utilizar el operador "*". Esto recogerá todos los argumentos posicionales no asignados en una tupla. Por ejemplo:

def indeterminados_posicion(*args):
    for arg in args:
        print(arg)

indeterminados_posicion(1, 2, 3, 4)  # Imprime 1, 2, 3, 4 en líneas separadas

En este caso, "args" es una tupla que contiene todos los argumentos posicionales que se pasaron a la función.

Para manejar un número indeterminado de argumentos de palabras clave (en inglés keyword args), podemos utilizar el operador "**". Esto recogerá todos los argumentos de palabras clave no asignados en un diccionario. Por ejemplo:

def indeterminados_nombre(**kwargs):
    for key, value in kwargs.items():
        print(f"{key} = {value}")

indeterminados_nombre(nombre="Alice", edad=25)  # Imprime "nombre = Alice" y "edad = 25" en líneas separadas

En este caso, "kwargs" es un diccionario que contiene todos los argumentos de palabras clave que se pasaron a la función.

Si queremos aceptar ambos tipos de parámetros simultáneamente, entonces debemos crear ambas colecciones dinámicas. Primero los argumentos indeterminados por valor y luego los que son por clave y valor:

def super_funcion(*args,**kwargs):
    total = 0
    for arg in args:
        total += arg
    print("sumatorio => ", total)
    for kwarg in kwargs:
        print(kwarg, "=>", kwargs[kwarg])

super_funcion(10, 50, -1, 1.56, 10, 20, 300, nombre="Hector", edad=27)

# Nos devolvería:

sumatorio => 390.56
nombre => Hector
edad => 27

Los nombres args y kwargs no son obligatorios, pero se suelen utilizar por convención. Muchos frameworks y librerías los utilizan por lo que es una buena practica llamarlos así.

Estas características hacen que Python sea muy flexible al manejar funciones con un número variable de argumentos.

¿Qué es una función de Python Lambda?

Una función Lambda en Python es una pequeña función anónima (no se les da un nombre explícito) que se define con la palabra clave "lambda". Estas funciones pueden tener cualquier número de argumentos, pero solo una expresión, y se pueden usar donde se requieran objetos de función.

La sintaxis de una función lambda es la siguiente:

lambda argumentos: expresión

Por ejemplo, la función lambda que suma dos argumentos podría ser:

sumar = lambda a, b: a + b

Aquí, 'a' y 'b' son los argumentos a la función, y 'a + b' es la expresión que se evalúa y se devuelve. Entonces, si llamamos a la función 'sumar' (como lo haríamos con cualquier otra función) con dos números:

print(sumar(6, 3)) # Imprime: 9

Otro ejemplo, puedes escribir una función lambda que duplique sus argumentos lambda x: x * 2 y usarla con la función map para duplicar todos los elementos de una lista:

mi_lista = [1, 2, 3, 4, 5, 6]
lista_nueva = list(map(lambda x: x * 2, mi_lista))
print(lista_nueva)  # [2, 4, 6, 8, 10, 12]

También puedes escribir una función lambda que revise si un número es positivo, lambda x: x > 0, y usarla con la función filter para crear una lista de números positivos.

mi_lista = [18, -3, 5, 0, -1, 12]
lista_nueva = list(filter(lambda x: x > 0, mi_lista))
print(lista_nueva) # [18, 5, 12]

También es posible que una función devuelva una función lambda.

Si necesitas funciones que, por ejemplo, dupliquen, tripliquen, etc... una función lambda puede ser útil:

def multiplicar_por (n):
    return lambda x: x * n

duplicar = multiplicar_por(2)
triplicar = multiplicar_por(3)
diez_veces = multiplicar_por(10)

duplicar(6)
12
triplicar(5)
15
diez_veces(12)
120

Las funciones lambda son útiles cuando necesitamos una pequeña función para un uso de corta duración, a menudo dentro de funciones, como por ejemplo, como argumento para funciones de orden superior (map, filter, etc.) que toman otras funciones como parámetros. También son útiles para definir funciones simples de una línea que se utilizan sólo una vez en nuestro código.

Usando una función lambda el código es más compacto y legible.

¿Qué es un paquete pip?

PIP (Pip Installs Packages) es un sistema de gestión de paquetes utilizado para instalar y administrar paquetes de software escritos en Python. Los paquetes son módulos de código que pueden ser reutilizados, y están disponibles públicamente a través del Python Package Index (PyPI), un repositorio (lugar donde se guarda algo) de software para el lenguaje de programación Python. Estos paquetes pueden ser instalados en cualquier sistema operativo que soporte Python, como Windows, macOS o Linux.

PIP proporciona una forma sencilla de instalar paquetes, permitiendo a los usuarios acceder y utilizar rápidamente el código escrito por otros desarrolladores.

Para comprobar si pip está instalado, abre una ventana de terminal y escribe el siguiente comando:

pip --version

Si no está instalado, puedes instalarlo con el siguiente comando:

python3 -m pip install --user pip

que instalará pip en tu directorio de usuario.

Si, por ejemplo, quieres instalar el paquete de manipulación de datos llamado Pandas (o el paquete requests, que proporciona utilidades para realizar solicitudes HTTP), simplemente puedes abrir tu terminal y escribir:

pip install pandas

PIP descargará e instalará el paquete Pandas y todas sus dependencias, haciendo que estén disponibles para su uso en tus programas Python.

Además de instalar paquetes, PIP también ofrece otras características para ayudarte a gestionar tus paquetes Python de manera eficiente, como la posibilidad de desinstalar paquetes que ya no se necesitan(1), actualizar un paquete a la versión más reciente(2), listar los paquetes instalados y sus versiones(3), mostrar información detallada sobre un paquete específico, como su versión, su ubicación, sus dependencias, etc.(4) y más.

(1)pip uninstall nombre_del_paquete
(2)pip install --upgrade nombre_del_paquete
(3)pip list
(4)pip show nombre_del_paquete

Los entornos virtuales son entornos de Python aislados que permiten instalar paquetes para que los use una aplicación en particular, en lugar de instalarlos en todo el sistema. Esto puede ser útil para evitar conflictos entre paquetes o para mantener versiones diferentes de un mismo paquete para diferentes proyectos.

Por ejemplo, para crear un entorno virtual llamado my_env, escribe el siguiente comando:

python3 -m venv my_env

Una vez que hayas creado un entorno virtual, debes activarlo antes de poder instalar paquetes en él. Por ejemplo, para activar el entorno virtual my_env, escribe el siguiente comando:

source my_env/bin/activate

Cuando el entorno virtual esté activado, verás el nombre del entorno virtual en el prompt de tu terminal y podrás trabajar con él.

Cuando hayas terminado de usar el entorno virtual, puedes desactivarlo escribiendo el siguiente comando:

deactivate

Puede decirse que pip es una herramienta rápida, sencilla e indispensable para cualquier desarrollador Python.
